# Add Rep or Weight

A small web app to preview 1RM (one-rep max) scenarios: enter current weight and reps, choose a 1RM formula, and see estimated 1RM results for incremental weight changes.

# Development
Install dependencies:

```bash
npm install
```

Start the dev server (hot reload):

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
